# 📊 Data Analytics Customer_Behavior_Analysis Project

## 🔎 Overview

This project demonstrates an end-to-end data analytics workflow  from data loading and exploration to dashboard creation and reporting. The objective is to extract meaningful insights from raw data and present them in a clear, business-friendly format.

## ⚙️ Project Steps

### 1️⃣ Data Loading

* Imported dataset using Python
* Checked data structure, datatypes, and missing values

### 2️⃣ Data Cleaning

* Handled null values
* Removed duplicates
* Standardized formats
* Created derived columns

### 3️⃣ Exploratory Data Analysis (EDA)

* Analyzed trends and patterns
* Identified key performance indicators (KPIs)
* Visualized distributions and relationships

### 4️⃣ SQL Analysis

* Wrote SQL queries for:

  * Aggregations
  * Joins
  * Subqueries
  * Window functions
* Created analytical tables for reporting

### 5️⃣ Dashboard Creation

* Built an interactive **Power BI dashboard** including:

  * KPI Cards
  * Trend charts
  * Filters & slicers
  * Category-wise performance visuals

### 6️⃣ Reporting & Presentation

* Summarized insights in a structured report
* Created a presentation using **Gamma** highlighting:

  * Business insights
  * Recommendations
  * Data-driven decisions
